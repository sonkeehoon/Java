숙명 vs 운명

Error(숙명)과 Exception(운명)의 차이

Error : 우리가 만든 문제가 아닌, 프로그램이 동작하는 환경의 문제일때 발생

Exception : 내가 짠 코드가 의도와 다른 상황에 직면할때 발생(예외적인, 예상하지 못했던 상황)
